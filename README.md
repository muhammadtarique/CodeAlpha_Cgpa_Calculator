# CodeAlpha_Cgpa_Calculator
A console-based CGPA Calculator that computes Semester GPA and Overall CGPA.
Developed as part of my internship at CodeAlpha.

## Features
- Takes multiple courses as input (course name, grade, credit hours).
- Converts letter grades (A, A-, B+, B, C+, C, D, F) into grade points.
- Calculates Semester GPA using total quality points / total credits.
- Optional: add previous CGPA and previous completed credits to compute updated overall CGPA.
- Displays a summary of all entered courses with grades and credit hours.
